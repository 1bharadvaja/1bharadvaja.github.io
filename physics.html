First Problem You'll Be Tackling

A pen of unit length is lying tilted on a rough table.

The lower end of the pen is dragged in the horizontal direction until the pen becomes vertical. During this process, the upper end of the pen travels a unit distance.

How many units does the lower end travel?

Assume that the velocity of the upper end of the pen is always parallel to the pen.

Answer

First thing we can do, is draw a mini diagram of this.

In this picture, we need to find x, which is simple found by integrating dx, but we dont know the full bounds of integration.

We can substitute ${\mathrm{d}x}$ for $\csc\theta {\mathrm{d} \theta }$. We then integrate $\int_{\theta =a}^{\pi \over2}\csc\theta {\mathrm{d} \theta }$. We do not know the lower bound of the integral, therefore we need to find it by using another integral.

We have to use another integral, at this point, to find the lower bound for the integral. We know that the top of the pen moves 1 unit. This means P = 1. You can set 1 = $\int_{a}^{\pi \over 2}{\mathrm{d}p} = \int_{a}^{\pi \over 2} \cot \theta {\mathrm{d} \theta } $. This is because in the diagram, $\cot \theta =$${\mathrm{d} p} \over {\mathrm{d} \theta }$, so therefore $\cot \theta {\mathrm{d} \theta } = {\mathrm{d} p}$.

$\int_{a}^{\pi \over 2} \cot \theta {\mathrm{d} \theta } = \ln |\sin \frac{\pi }{2}|-\ln |\sin \alpha |$

$\ln |\sin \frac{\pi }{2}|-\ln |\sin \alpha | = 1$. Then you can solve, $\ln |\sin \frac{\pi }{2}| = 0$ and $0-\ln |\sin \alpha | = 1$

Since $-\ln |\sin \alpha | = 1$, $e^{-1} = \sin \theta$, so therefore, $\theta = \arcsin (\frac{1}{e})$. Now we have found out the lower bound.

Now we can evaluate the first integral. $\int_{\arcsin (\frac{1}{e})}^{\pi \over2}\csc\theta {\mathrm{d} \theta }$.

This evaluates to $-\ln|\csc \frac{\pi}{2} + \cot \frac{\pi}{2}| + \ln|\csc (\arcsin (\frac{1}{e}))+ \cot (\arcsin (\frac{1}{e}))|$. At this point, we can draw a triangle to figure out the values.
