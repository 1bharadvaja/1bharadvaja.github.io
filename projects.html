XprAI

This was a submission to LingHacks 2019. Me and my teammate basically used OpenCV, and also combined NLTK with a Tensorflow ML model to determine sentiment. We created a program where a bot asks questions to a person, while running OpenCV in the background to detect someone's emotions (using a simple function we built with haarcascades). Through this process, it determines, if you seem to be happy, or if you seem to be sad, depressed, or even suicidal. Below is an image of HaarCascades in action when the program is running.
