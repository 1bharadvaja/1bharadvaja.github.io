Learn Swift

Swift is a language for programming iOS apps. The previous iOS programming language, Objective-C, is long gone.

Swift has 3 main things you need to use. There are things called variables, which you initialize by saying "var someVar", which means the value can change, or saying "let myLetVar".

To make it so that you don't have any errors, when you declare the variable, you nheed to give it a type. Ex. "let myLetVar: String" The types are Double, Int, Float, String. You declare based on what the variable is used for. And then, If you want to declare what the variable will store straight away, you will need to do "var myVar: String = "Hello World".

Print statements print what you want to say to the console. For example you can take the variable we made, and say "print(myVar)" which will print out "Hello World"!

Another type that is important in Swift are methods. Methods are the same as functions. To initialize a function, say "func myFunc()"
